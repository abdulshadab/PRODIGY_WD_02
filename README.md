# ⏱️ Stopwatch Web Application

## 📖 Overview
This project is a fully functional **Stopwatch Web Application** developed using HTML, CSS, and JavaScript. It allows users to accurately measure time intervals with features such as start, pause, reset, and lap tracking.

The application focuses on delivering a clean user interface along with real-time interactivity, making it both user-friendly and efficient.

---

## ✨ Features
- ▶️ Start the stopwatch  
- ⏸️ Pause/Resume functionality  
- 🔄 Reset the timer  
- 🏁 Record and display lap times  
- ⏱️ Real-time time tracking  
- 🎨 Simple and responsive UI  

---

## 🛠️ Tech Stack
- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Logic and interactivity
